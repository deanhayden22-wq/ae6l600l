# Patches - EXPERIMENTAL

This directory contains experimental patches for the AE5L600L ROM. These are **unverified and may contain incorrect addresses or logic**.

- `cluster-boost-psi/` - Cluster boost PSI display patch (experimental)
- `tgv-ethanol-sensor/` - TGV delete / ethanol sensor patch (experimental)

Do not treat these as validated work. Cross-reference against verified disassembly before using.
