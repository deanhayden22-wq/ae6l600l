================================================================================

startup infrastructure, RTOS kernel, ISR dispatch system, and hardware
peripheral initialization code.

================================================================================
SECTION 6: ARCHITECTURAL NOTES
================================================================================

6.1  Boot Sequence
──────────────────
  1. Power-on reset → vec[0] → 0x000C0C (reset_entry_point)
  2. hw_init_A: WDT setup (F70A), bus state controller
  3. hw_init_B: ATU timer base setup (F720-F726)
  4. RAM clear (FFFF0000-FFFFBFFF zeroed)
  5. BSS init + static data copy
  6. Peripheral init chain: 0x012600-0x01E3FF functions
  7. RTOS scheduler start → task10 → all other tasks

6.2  ISR Dispatch Architecture
──────────────────────────────
  1. Any peripheral interrupt → NMI vector → 0x000BAC
  2. Context save (all regs including FPU)
  3. 0x000EE4 (intevt_resolver): reads INTEVT register (FFFF70B)
     Maps source to class 0/1/2/FF
  4. 0x000F4C (dispatch_manager): indexed jump via table at 0xE5EC
     54 handler slots, each a function pointer
  5. Handler executes → RTE → context restore

6.3  RTOS Task Architecture
───────────────────────────
  - 13 task slots in master_task_table at 0x003F80
  - Each entry: 16 bytes (priority, period, state, entry address)
  - Scheduler tick from ATU interval ISR (0xE970)
  - Task epilogue at 0x0035A4: common exit path
  - Sub-task dispatchers at 0xF97C and 0x10D58: secondary tables
    for tasks 2/3/12 that iterate additional function arrays

6.4  The 0x012600-0x01E3FF Block in Context
───────────────────────────────────────────
  This ~48KB block is the SH7058 BSP (Board Support Package) —
  hardware peripheral initialization code that:
  - Configures all GPIO pins (input/output/alternate function)
  - Sets up ATU timer channels for injection timing
  - Initializes ADC converter channels and scaling
  - Configures serial interfaces (SCI, CAN)
  - Sets interrupt priorities and enables
  - Calibrates sensor input chains (voltage → physical units)

  This code runs once at startup and is largely static across ROM
  revisions. It is NOT tuneable and does not contain calibration
  parameters relevant to engine tuning. The 213 functions here are
  the standard Denso initialization library for the SH7058 platform.
